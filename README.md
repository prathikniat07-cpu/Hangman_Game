A simple and interactive Hangman Game built using Python 🐍<br/>
This project randomly selects a word from a predefined word list, and the player has to guess the word letter by letter before running out of lives ❤️<br/><br/>

✨ Features<br/>
🎲 Random word selection<br/>
🖼️ ASCII Hangman art for each life stage<br/>
✅ Tracks correct guesses<br/>
🚫 Prevents repeated guesses<br/>
🏆 Win/Lose conditions<br/>
👨‍💻 Beginner-friendly Python project<br/><br/>

🛠️ Technologies Used<br/>
🐍 Python<br/>
🎯 Random Module<br/><br/>

📂 Project Structure<br/>
📄 solution.py → Main game logic<br/>
📚 hangman_words.py → Word database<br/>
🎨 hangman_art.py → Hangman ASCII art and logo<br/><br/>

▶️ How to Run<br/>
💻 python solution.py<br/><br/>

📖 What I Learned<br/>
🔁 Python loops and conditionals<br/>
🧩 Working with lists and strings<br/>
📦 Importing modules<br/>
🧠 Game logic implementation<br/>
⌨️ User input handling<br/><br/>

🚀 A fun beginner Python project that helped me strengthen my programming fundamentals and logical thinking.
